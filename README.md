# Morce Cube

![MorceCube image art](assets/cube.png)

A simple and beautiful terminal-based application which generates scrambles for your Rubik's Cube and runs the timer for your solves, storing your times and the scrambles in a file.

## Requirements
`bash 5.1` must be installed in your system.

Optionally, you can install `ascii-image-converter` to activate the splash image screen of `MorceCube`.
Download the binary file corrresponding to your system, available in `https://github.com/TheZoraiz/ascii-image-converter/releases/tag/v1.13.1`.

## Setup
Clone this repository, enter the created directory, and run the `cubeme` executable in your terminal.

Enjoy your Cube!
